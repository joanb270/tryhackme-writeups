## ğŸ”´ Offensive Security Overview

### What is Offensive Security?

Offensive security focuses on **finding and exploiting vulnerabilities** before malicious attackers do. It simulates real-world attacks in a **legal and controlled environment**.

### Key characteristics

- Think like an attacker
- Identify weaknesses in systems, networks, and applications
- Exploit vulnerabilities to demonstrate impact
- Report findings so they can be fixed

### Common Offensive Roles

- Penetration Tester
- Ethical Hacker
- Red Team Operator

### Why Offensive Security Matters

Understanding how attacks work helps organizations improve defenses and reduce risk.

## ğŸ”µ Defensive Security Overview

### What is Defensive Security?

Defensive security focuses on **protecting systems, detecting attacks, and responding to incidents**.

Instead of breaking in, defenders focus on **monitoring, analysis, and response**.

### Core Defensive Domains Introduced

- **SOC (Security Operations Center)**
- **DFIR (Digital Forensics & Incident Response)**
- **Threat Intelligence**
- **Malware Analysis**
- **SIEM Monitoring**

### Typical Defensive Tasks

- Monitoring alerts
- Analyzing logs
- Investigating suspicious activity
- Responding to incidents
- Improving detection rules
